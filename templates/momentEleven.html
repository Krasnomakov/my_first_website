Outcome #11

Website Blueprint

PROFESSIONAL SKILLS: targeted interaction

TECHNICAL SKILLS: iterative process, interacive prototypes

Welcome to Unio Room, a game event and game device rental website that was created by team Tetra Lenses. Take a look around and explore the features and functionality of the website. We received feedback on the overall design, user experience, and received suggestions for improvements during the presentation. Overall, users loved the design and it was easy to navigate and understand how to use the website. However, there is a room for improvement in terms of making events better presented. Also, structure elements and design could be aligned better if we had more time. That requires more user testing that we aim to conduct in the future.
